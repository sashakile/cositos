# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in cositos, please report it privately
by opening a GitHub Security Advisory at:

https://github.com/sashakile/cositos/security/advisories/new

Please do **not** report security vulnerabilities through public GitHub issues.

The maintainer will acknowledge receipt and work on a fix as capacity permits.

## Scope

This security policy covers the `cositos` Python package, its Julia, R, C#,
and Clojure backends, and the documentation site. Issues in dependencies
(ipykernel, IJulia, etc.) should be reported to the respective project.
